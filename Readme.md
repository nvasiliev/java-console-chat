### Java Console Chat

Asynchronous chat server based on java NIO and sockets

#### Development

Go to project root

##### Build

```
gradle build
```

##### Run

```
java -jar build/libs/java-console-chat-all.jar
```

##### Connect to the server

While client development is WIP use `telnet` instead:

```
telnet 0.0.0.0 4444
```
